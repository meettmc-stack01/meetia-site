# IMチェックのチャートでわかること

## 記事の役割

IMチェックの結果がチャートで表されること、そのチャートから何を読み取るのかを説明する記事。

## 元記事から使える内容

- 8本のボトルを使って心と体の状態をグラフ化すること
- 結果は視覚的にとらえやすいチャートグラフに変換されること
- チャートは状態を知るだけでなく、理想的な状態に近づけるための選択肢を考える地図になること
- 食事、運動、入浴、休日の過ごし方、アロマセルフケアなどへの活用

## 初稿メモ

IMチェックでは、8種類の香りを好きな順に並べ、その結果をチャートに表します。

チャートは、今の状態を「良い・悪い」で判断するものではありません。自分の傾向を知り、どのように整えていくと心地よく過ごしやすいかを考えるための手がかりです。

日常の中では、食事、休み方、運動、入浴、アロマテラピーを使ったセルフケアなどに活かすことができます。
