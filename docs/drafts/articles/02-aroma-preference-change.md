# 香りの好き嫌いが変わる理由

## 記事の役割

香りの好みが固定されたものではなく、その時の心身の状態によって変わることを説明する記事。

## 元記事から使える内容

- 香りを心地よいと感じるか、嫌だと感じるかは生存欲求からのメッセージとして捉えられること
- 自分の体に足りないもの、要らないものを、香りを使って自分に尋ねるという考え方
- 香りの反応を通じて、自分の状態を知ることができるという説明

## 初稿メモ

同じ香りでも、ある日は心地よく感じ、別の日にはあまり好きではないと感じることがあります。

嗅覚反応分析では、その変化を単なる気分の違いとして片づけず、今の心身の状態を知る手がかりとして見ていきます。

香りの好き嫌いは、「今の自分にとって必要か」「少し負担に感じるか」を知るための入口になることがあります。
